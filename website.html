Berita Website Terbaru

Hai pembaca...

 Ini merupakan website pertama saya. Sebenarnya ini merupakan tugas dari dosen Pemrograman Berbasis Web di kelas saya. Saya sendiri baru belajar membuat web dengan HTML dan CSS di Politeknik Statistika STIS, dimana merupakan tempat saya berkuliah sekarang. Fyi guys, HTML sendiri adalah singkatan dari Hypertext Markup Language. Bahasa HTML digunakan untuk membangun halaman web. Selain itu, HTML adalah bahasa markup yang umum digunakan oleh para developer web. Karena kemudahan dalam menggunakannya, HTML menjadi bahasa pemrograman web yang populer dikalangan para developer.
  HTML mempunyai fungsi di antaranya dapat menentukan format suatu teks, membuat list, membuat link ke dokumen lain, menyisipkan gambar, serta dapat menampilkan informasi dalam bentuk tabel. Selain itu, HTML juga berfungsi untuk mengatur tampilan dari halaman web dan isinya, menambahkan objek seperti gambar, video, audio, dan lain-lain. Sedangkan untuk CSS (Cascading Style Sheet), itu merupakan salah satu bahasa desain web (style sheet language) yang mengontrol format tampilan sebuah halaman web yang ditulis dengan menggunakan penanda(markup laguage. Biasanya CSS digunakan untuk mendesain sebuah halaman HTML dan XHTML, tetapi sekarang CSS bisa diaplikasikan untuk segala dokumenXML, termasuk SVG dan XUL bahkan ANDROID. Jika kalian semua mau belajar (seperti saya), saya rekomendasikan W3school atau W3bai untuk mempelajari mulai dari dasar cara untuk pemrograman (tidak hanya membuat web).
